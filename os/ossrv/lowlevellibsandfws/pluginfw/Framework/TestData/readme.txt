
The SPI ecom-0-2.s02 and ecom-0-3.s03 are needed for Test Case-SYSLIB-ECOM-CT-3170(Enable Dynamic Language Switching in ECOM on Z: drive)

ecom-0-2.s02 is the SPI for rsc file languageplugin2.r02, created using cmd spitool.pl -tecom-0-2.s02 -d./ LanguagePlugin2.r02

ecom-0-3.s03 is the SPI for rsc file languageplugin2.r03, created using cmd spitool.pl -tecom-0-3.s03 -d./ LanguagePlugin2.r03
